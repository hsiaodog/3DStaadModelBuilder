節點編號(全局)

編號預覽:—
編號規則說明
軸座標排序
每軸 unique 座標依數值升序 → rank 1, 2, 3, … 編號格式由「編排優先」決定:
水平優先 → XX·ZZ·YY,垂直優先 → YY·ZZ·XX
節點分類(影響 ID 是否加 offset)
▸ Anchor(正常編號,任一滿足):
① 手動標 ▼錨點(選取節點後按「設為錨點」)
② 連接桿件 < 2 條(柱腳、樑端、孤立點)
③ 任一對連接桿件互相垂直(2D 90°±3°,含斜桿之間)
④ 純連續直桿件(所有方向向量共線,例:2+ V 共線無分岔)
⑤ 跨三平面 globalJoint(XY + YZ + XZ 都有 binding)
▸ Demote(ID 加 offset 推到後段,全滿足):
≥ 2 條 member,沒垂直配對,不是純連續直桿,沒跨三平面,沒手動 anchor。
典型:純斜撐端點、連續直桿+斜撐(brace 在柱中段)
offset = 10Nx+Ny+Nz。例:5 位數編號 → demote 後 ID 加 100000,範圍 100000~199999。
精準度
節點適配位數 = bucket 大小;兩節點顯示同數值 → 同 rank。所有 globalJoint 座標 round 到此精準度。

Plane:XY → YZ → XZ
